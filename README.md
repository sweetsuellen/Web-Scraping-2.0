Web Scraping 2.0
Este projeto é uma ferramenta de automação desenvolvida em Node.js para otimizar a extração de dados cadastrais de empresas. O objetivo principal é eliminar tarefas manuais repetitivas e acelerar a rotina de prospecção (LDR).

O que esta ferramenta faz?
O script realiza consultas automatizadas em múltiplas fontes oficiais (ReceitaWS, Portal da Transparência e Comprasnet), cruzando as informações para entregar um resultado consolidado em uma única planilha Excel.

Principais funcionalidades:

Coleta de dados básicos (razão social, endereço, contato).

Extração de dados complementares, incluindo histórico de sanções, recursos recebidos e vigência de contratos.

Geração de arquivos Excel estruturados e prontos para análise comercial.

Otimização significativa do tempo de produção por lead.

Como utilizar
Certifique-se de ter o Node.js instalado em sua máquina.

Instale as dependências necessárias.

Insira a lista de CNPJs no arquivo de entrada.

Execute o script e aguarde a geração do arquivo final.
